# uestc-tvly-web

> IPTV client for UESTC.

## Documentation

see [here](docs/).

## Development

``` shell
yarn install
```

### Run serve

``` shell
yarn dev
```

### Run build

``` shell
yarn build
```

### Run lint

``` shell
yarn lint
```
